Cores

Existem várias formas de definirmos cores utilizando o CSS, elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl (hue, saturation, ligtness)
  • Hue (matriz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Ligthness (luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
